# 올바른대기열 — 개발 해야 할 일 전체 목록

> 최종 업데이트: 2026-04-09

---

## 솔루션 프로세스 (확정)

```
1. 환자 태블릿 접수 → [pending]
2. 인포직원 확인 → 의사랑 접수
3. 인포직원 "접수완료" → [confirmed]
4. 환자에게 문자/카카오 알림 (대기순서 + 링크)
5. 환자 링크로 대기순서·예상시간 확인
6. 진료 순서 임박 시 알림
7. 인포직원 "진료중" → [in_progress]
8. 인포직원 "진료완료" → [completed]
```

## 미연결 기능 (기존 코드 수정)

| # | 작업 | 현재 상태 |
