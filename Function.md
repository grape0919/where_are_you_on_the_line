## 기능 체크리스트

### 완료된 기능
- [x] 회원 등록/대기열 토큰 발급 및 폼 검증
- [x] 대기열 페이지 ETA 표시, 갱신, 토큰 오류 처리, 임박 알림 강조
- [x] 예약 흐름: 환자 정보 검증, 슬롯/일자 제한, 요약 복사·공유
- [x] 관리자 대시보드: 목록 조회, 삭제/완료 확인, 통계 표시
- [x] 서비스/의사/환자/설정/예약 정원 관리(로컬스토리지 기반)
- [x] CSV 업로드로 진료항목별 예상 대기시간 자동 산출·적용(동적 헤더, 통계 전략)
- [x] 환자 ID 정규화/자동 생성 및 방문 이력 확인
- [x] 예약 한도 기본값 및 알림 문구 환경변수 적용
- [x] Vitest + JSDOM 테스트 실행 환경(.env.test)과 문서 정리
