## 기능 체크리스트

### 완료된 기능

- [x] 접수 및 대기표(토큰) 발급, 잘못된 입력 안내
- [x] 대기 현황: 예상 대기시간 표시, 자동 새로고침, 오류 안내, 호출 임박 강조
- [x] 예약: 정보 확인, 날짜·시간 제한, 결과 복사/공유
- [x] 관리자 화면: 목록 조회, 삭제/완료 확인, 기본 통계
- [x] 진료 항목·의사·환자·설정·예약 정원 관리(내부 저장소 사용)
- [x] CSV 업로드로 진료 항목별 예상 대기시간 계산·적용(헤더 자동 인식, 통계 방식 적용)
- [x] 환자 번호 자동 생성·정리 및 방문 이력 확인
- [x] 예약 한도 기본값과 안내 문구를 환경설정으로 제어
- [x] 테스트 실행 환경(.env.test)과 가이드 정리

### 구현 예정/추가 개선

- [ ] 관리자 인라인 수정 시 화면을 바로 반영하는 방식
- [ ] 관리자 로그인/권한 확인과 API 보호
- [ ] 저장소를 더 안정적인 서버 저장소(예: Redis/DB)로 이전
- [ ] 예약·대기 현황을 실시간으로 반영(예: Redis 활용)
- [ ] 데이터베이스 스키마와 마이그레이션으로 기록·계정 관리
- [ ] CSV 업로드 예시 템플릿 제공
- [ ] 추가 통계(최대값/표준편차 등)와 통계 방식 선택 기능
- [ ] 서비스 분리 시 보안·CORS·헬스체크 등 운영 강화
